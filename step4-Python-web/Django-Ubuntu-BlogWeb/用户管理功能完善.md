### *判断用户登录状态以及注册、退出：*
+ **下载django**
    ```shell
    pip3 install django
    ```
    ```shell
    # 如果还没有安装pip3：
    sudo apt-get install python3-pip
    ```
+ **建立django项目**
    ```django
    # 在项目文件夹下执行指令
    django-admin startproject crawlweb
    ```

    但是！执行后会出现下面报错：
    ```shell
    zsh: command not found: django-admin
    ```
